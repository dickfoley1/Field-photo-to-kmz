# GitHub Upload Steps

## Create The Repository

1. Sign in to GitHub.
2. Click `New repository`.
3. Name it something like `field-photo-to-kmz-studio`.
4. Leave the repository empty.
5. Click `Create repository`.

## Upload This Project

1. Open the repository page on GitHub.
2. Click `uploading an existing file`.
3. Drag in the contents of the final prepared upload folder.
4. Commit the upload.

## What Other Users Should Do

1. Download the repository ZIP from GitHub.
2. Extract it.
3. Run `setup_windows.bat`.
4. Run `launcher.bat`.
5. Optional: run `create_desktop_shortcut.bat`.

`setup_windows.bat` will also extract `exiftool_bundle.zip` if it is included in the repo.
